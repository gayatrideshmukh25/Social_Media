A full-stack **Social Media Web Application** that allows users to create posts, like or dislike posts, update content, and manage user profiles.  
Built using **React + Vite** for the frontend and **Node.js + Express** for the backend with MVC architecture.

## Project Structure

my-project/
├── frontend/ # React + Vite
├── backend/ # Node + Express
└── README.md

## Setup Instructions

```bash
cd frontend
npm install
npm run dev



cd backend
npm install
npm run dev



```
